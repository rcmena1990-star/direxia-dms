# DIREXIA DMS

## Sistema Integral de Gestión Empresarial

**DIREXIA DMS** es una plataforma empresarial desarrollada por **DIREXIA CONSULTING** para la gestión integral de:

- CRM
- Clientes
- Proyectos
- Entregables
- Finanzas
- KPIs
- Gestión Documental
- Sistemas Integrados de Gestión (SIG)
- Homologaciones
- Portal del Cliente
